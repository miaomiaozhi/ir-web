2023年寒假英国牛津大学展望计划线上项目报名通知
国际交流与合作部　2022/11/23 16:46:00
 
一、项目介绍
我校与英国牛津大学摄政学院（Regent's Park College）合作开展“牛津展望计划（Oxford Prospects Programmes）”。为了使同学们能够在疫情期间持续利用高质量的学习资源和学术服务，该项目线上进行。牛津大学摄政学院全球发展与展望研究院（OPGDI）将于2023年寒假继续开设在线跨学科课程，课程各模块由对应的英国四大学术院（国家学术院、皇家学会、皇家工程院，医学科学院）的院士领衔，顺利完成课程的学生将获得结业证书与成绩单。
对于学有余力的同学（高年级或者自身英语能力较强，对于注册访问生项目感兴趣的学生），牛津大学另设了为期9周的导师制（Tutorial）项目（牛津大学最核心的授课模式）。导师项目以牛津大学导师辅导制（Tutorial System）为依托，学生将以2-3人小组形式，接受牛津导师的针对性学科辅导，以促进学生个体的认知、发展和提升，培养独立思考及批判性思维能力。时间为：2023年3月-5月，（申请者可同时申请跨学科学术课程及导师辅导制课程，也可选其一申请，请在申请时按要求进行备注）
 二、课程安排
1. 学术课程
共计15学时，分六大类跨学科Module，选其一参加：
Module A: 政治-经济-哲学-法律；
Module B: 文学-语言-数字文化与传播学；
Module C: 可持续发展与气候变化：地理-海洋-环境-能源；
Module D: 金融-商科-管理；
Module E: 前沿科学：数学-物理-计算机-工程；
Module F: 医学（仅供医学类专业学生申请）。
2. 小班研讨会
共计10学时，每日学术课程之后将安排线上互动研讨课，学生将分为小组（2-4人），对学术内容进行深度讨论，课上作业展示。
3. 拓展工作坊
共计5学时，旨在锻炼学生批判性思维和研究技能，明晰学术和职业规划，同时还将提供与牛津大学成功申请者互动，建立新人际关系的机会。
4. 特邀嘉宾讲座
共计三场次，将邀请来自于不同行业的重量级嘉宾，为同学们带来行业内的洞见和思考。A. 影视戏剧行业《唐顿庄园》总制片人；B. 世界顶级企业组织高管；C. 英国自然纪录片制作团队。
 更多项目信息请参见文末附件。
 三、报名条件
1. 我校全日制在校本科生和硕士生；
2. 学生需对所报名参加的领域有相应背景知识及强烈的学术兴趣；
3. 外语水平要求：雅思6.0或托福 80。
注：如在报名时没有以上英语成绩证明，可提供其他英语能力证明（如四级、六级、或高考成绩等），牛津大学项目学术处将依据申请人资历进行审核，安排面试，OPGDI学术处将对录取结果拥有最终决定权。
 四、时间安排
1. 课程时间：
Module A, Module B, Module C： 2023年1月9日-1月20日；
Module D, Module E, Module F： 2023年1月30日-2月10日；
2. 报名截止时间：2022年11月30日。
 五、项目费用
 跨学科课程原费用为1,650英镑，我校学生经减免后实缴费为：1,050英镑；
导师制课程费用实缴：1260英镑。
 六、报名方式
1. 参加项目的同学填写2023 Oxford Online Programme 申请链接https://jinshuju.net/f/XMpmPI；
2. 同时需填写附件申请表进行校内报名，可以待牛津录取后再交学院签字盖章版申请表到国际部（汇元楼328，或扫描发szufao@szu.edu.cn）；
3. 需提供英语水平证明材料（IELTS、TOEFL等）。
